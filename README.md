## Project Description

**Smart Traffic Management System** is a **C++ console project** for managing urban traffic. It uses **graphs** to model roads and intersections, allowing:

* Traffic simulation for multiple vehicles
* Emergency vehicle routing
* Road management (insert/block)
* Congestion analytics
* Traffic signal monitoring

All data is handled via **CSV files** for persistence and easy modification.

---

## Technologies

* **Language:** C++
* **Data Structures:** Graph, Linked List, Priority Queue, Max Heap
* **File Handling:** CSV files for network, vehicles, signals, and closures

---

## Usage

1. Launch the app and select options from the main menu:

   * Network & Infrastructure
   * Traffic Operations
   * Emergency & Analytics
   * System Logs & Settings
2. Perform operations like simulation, routing, or network updates.
3. Save changes before exiting.

---

## Algorithms

* Dijkstra for shortest path
* A* / Priority routing for emergencies
* Max Heap / Priority Queue for congestion & vehicle priorities
